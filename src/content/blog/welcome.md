欢迎来到我的个人空间！

这是一个使用 [Astro](https://astro.build) 构建的极简风格网站。

## 特性

- 🚀 极简设计
- 🌙 支持暗色模式
- 📝 Markdown 博客支持
- ⚡ 静态生成，高性能

## 快速开始

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 构建生产版本
npm run build
```
